Data pack 3 / Track data

1E6BDCommon tile graphics offset - upper 8 bits
1E6C0 -> 1E6C1Common tile graphics offset - lower 16 bits
1E749 -> 1E74BAddress to track indexes
Unknown data (!)
1EB4B -> 1EB5ATile type index for each theme (2 bytes per theme)
(Tile type data is at 7FDBA)
1EB5B -> 1EBA2Track map address index (3 bytes each)
To have the hex address equivalent, read the 3 bytes backward,
removing the 1st character of the 3rd byte.
Example, Mario Circuit 3: pointer = 008086, hex address = 68000
